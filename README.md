# Personal Checklist App

A daily/weekly/monthly checklist tracker built with FastAPI and SQLite.

## Features
- Daily task tracking (GYM, DSA, ML)
- Weekly task tracking (Django, SQL, Project Work)
- Weekly and Monthly reports
- History view
- Progress statistics

## Deployment on Vercel

This app is configured for deployment on Vercel.
